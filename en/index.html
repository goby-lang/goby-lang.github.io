Concurrency Support

In Goby you can open a new thread by just using thread method, and passing objects between thread using Channel object (see Thread and Channel for more details)

Builtin Multi-threaded Server

Goby has builtin multi-threaded web server called SimpleServer with sinatra-like interfaces. We have our sample site built upon it, go check it out!

Plugin System

Plugin System allows you to extend your Goby app with Go packages without writing any Go code

Quick Setup

$ brew tap goby-lang/goby
$ brew install goby

Try it in Interactive Mode

Just run

$ goby -i
